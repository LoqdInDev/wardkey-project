3. Description of Service

WARDKEY is a password manager with zero-knowledge vault encryption that helps you securely store and manage passwords, payment cards, secure notes, TOTP keys, API keys, software licenses, passkeys, and other sensitive data. Key features include:

  • Local-first storage: Your vault is encrypted and stored on your device by default
  • Optional cloud sync: Encrypted vault data can be synced across devices via our servers
  • Browser extension: Autofill and save credentials on websites
  • Password generator: Create strong, unique passwords
  • TOTP authenticator: Store and generate time-based one-time password codes
  • Security Monitor: Identify weak, reused, or old passwords
  • Breach scanner: Check your passwords against the Have I Been Pwned database using k-anonymity (only partial password hashes are sent)
  • Secure sharing: Share credentials via time-limited, view-limited encrypted links. Encryption keys are embedded in the URL fragment and never sent to our servers
  • Emergency access: Designate trusted contacts who can request access to your vault with a configurable waiting period
  • Import/Export: Import from other password managers (LastPass, 1Password, Bitwarden, Chrome) and export your vault data
  • AI-enhanced analysis (optional): Password Analyzer and Security Report powered by Anthropic's Claude AI send anonymized vault metadata (never passwords) to our servers. The Phishing Detector sends user-provided content (text or URLs you paste) for analysis
